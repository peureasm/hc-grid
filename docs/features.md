# Feature Record

HC Grid currently includes the following grid features.

## Core Grid

- Web Component Custom Element: `hc-grid`
- Shadow DOM style encapsulation
- Property-based `columns` and `rows` data injection
- Header, body, and empty-state rendering
- Safe text rendering
- TypeScript public types
- ESM and UMD library builds

## Data Interaction

- Row click selection
- Checkbox multi-selection
- Sort cycle: none, ascending, descending, none
- Cell click, row click, sort, selection, edit, action, and page events
- Client-side CRUD sample with query, create, read selected, update, and delete flows

## Column Features

- Width and min-width
- Alignment: left, center, right
- Sortable columns
- Resizable columns
- Formatter function
- Editable cells

## Built-In Cell Components

- Button
- Checkbox
- Dropdown menu
- Input
- Select
- Multi-select with checkbox popup
- Badge

## Layout

- Table view
- Card view
- Custom card renderer
- Card checkbox selection sample

## Pagination

- Client-side pagination
- Numeric pager with ellipsis
- Previous and next page controls
- Page-size select box
- Page change event

## Large Data

- Virtual scroll
- Configurable row height

## Export And Import Samples

- CSV export and download
- Excel-compatible export and download
- Excel upload sample

## Theming

- CSS variable based theming
- Light and dark theme examples
